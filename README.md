# Hi, I'm Lucas

我主要在做 AI 工作流、Agent 技能和内容工具。

我关心的不是“AI 能不能做出一个 demo”，而是这些东西能不能真的进入日常工作：能不能复用，能不能交付，能不能被别人拿起来继续用。

最近我会比较多地折腾这些方向：

- Codex / Claude Code 的 Skill 工作流
- 面向创作者的 AI 内容工具
- 把一次性的 prompt 沉淀成可复用系统
- AI 配图、文章结构化、发布流程自动化
- Agent 工作台、规则文件和项目迁移

我会把一些小实验开源出来。大部分项目都不会一开始就很完整，但我会尽量让它们有一个清楚的问题意识：这个工具到底在帮人省掉哪一段重复劳动。

## 最近在做

**Personal IP Illustrator**

一个帮创作者生成个人视觉 IP 的 Skill。

它不是头像生成器，而是尝试把“风格选择、参考图、候选 IP、文章配图位置判断、个人专属 Skill 导出”串成一个完整流程。

我用一篇 AI 工作流文章跑了第一轮案例：生成 `Circuit Guide` 这个电路向导，并从 8 个段落里选出 3 个真正值得配图的位置。

Repo: https://github.com/YOLOooooos/personal-ip-illustrator
